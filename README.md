# MCP-based-agent
## Langgraph, langchain, mcp and Agent orchestration

By using langgraph and mcp concepts, I have defined how the future of agentic commerce would look like from the engineering angle. 
#  Workflow : 
User wants some food ---> agent would recommend cusines and restaurants nearby locations ----> user selects -----> agent places the order .
The user is allowed to order multiple cusines from multiple restaurants at the same time and agent will handle it!.
